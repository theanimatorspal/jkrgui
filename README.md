# JkrGUI
A Cross Platform Vulkan Based Graphics library for Lua, C++.

| Platforms | Support |
|:----------|:-------:|
| Windows   |   ✅    |
| Android   |   ✅    |
| macOS     |   ✅    |


There are following projects using this library.
